## Core User

Roofing contractors, office managers, and sales reps who need a lightweight way to track sent estimates and follow-up actions.

## Problem

Sent quotes go stale because follow-up is inconsistent and hard to track.

## MVP

- quote list dashboard
- aging and stale indicators
- follow-up queue
- AI-assisted follow-up drafts
- quote activity log

## Success Criteria

- users can log quotes quickly
- users can see which quotes need follow-up today
- users can generate or copy a follow-up message in seconds
